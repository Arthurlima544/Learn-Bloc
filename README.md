# About the Project

The objective of this project is:
- Deal with barcode scanning in a real context aplication.

In this Project we have:
- ChangeNotifier and Provider for state management
- camera
- Barcode Scanning
- User Preferences
- Routes
- Google Authentication


## How to run

```cmd
cd myapp
flutter pub pub get
```
